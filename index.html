📱 Turn Your Web App Into a Mobile App

🚀 What is Median.co?

Median.co lets you convert your web applications into native mobile apps for iOS and Android without writing any additional code!

  • Turn your HTML, CSS, and JavaScript web app into iOS and Android apps
  • No need to learn Swift or Kotlin - your web code becomes your mobile app
  • Publish directly to the App Store and Google Play
  • Automatic updates - no app store review delays
  • Access mobile features like camera, GPS, notifications, and more
  • One codebase, two platforms - save time and money

🎯 Getting Started with Median.co

Step 1: Create Your Web App

Build a responsive web app using HTML, CSS, and JavaScript. Host it on GitHub Pages, Netlify, or any web server.

Step 2: Sign Up on Median.co

Visit median.co and create an account

Step 3: Add Your Web App URL

Provide the URL of your deployed web app. Median will wrap it in a native app shell.

Step 4: Customize & Publish

Add app branding, icons, and publish to the App Store and Google Play.

📲 Access Native Mobile Features

Use the Median API to access powerful mobile capabilities:

📷 Camera

Send push notifications to users

📱 Contacts

Share specific pages of your app

✨ Best Practices for Mobile Apps

📐 Responsive Design

Make sure your web app works perfectly on mobile screens. Use Flexbox, CSS Grid, and media queries.

⚡ Performance

Optimize images, minimize code, and lazy-load content to ensure your app loads quickly.

🎯 Touch Friendly

Make buttons and interactive elements large enough to tap easily on mobile devices (minimum 44x44 pixels).

🔄 Offline Support

Use Service Workers to enable offline functionality and improve reliability.

🔒 Security

Use HTTPS, validate user input, and protect sensitive data. Never store passwords in local storage.

🐙 GitHub & Deploy with GitHub Pages

📚 What is GitHub?

GitHub is a platform for version control and collaboration. It lets you:

  • Store your code in the cloud
  • Track changes with Git version control
  • Collaborate with other developers
  • Deploy your projects for free with GitHub Pages
  • Showcase your portfolio to employers

🚀 Getting Started with Git

1. Install Git

Download from git-scm.com

2. Create GitHub Account

Sign up at github.com

3. Create a Repository

Click "New Repository" on GitHub

💻 Essential Git Commands

Check Status

git status

Shows which files have changed

Add Changes

git add .

Stages changes for commit

Commit Changes

git commit -m "Add new feature"

Saves changes with a message

Push to GitHub

git push origin main

Uploads changes to GitHub

🌍 Deploy Your Website for Free!

Host your website using GitHub Pages in 4 simple steps:

Step 1: Create index.html

Make sure your main HTML file is named index.html

Step 2: Push to GitHub

git add . git commit -m "Initial commit" git push origin main

Step 3: Enable GitHub Pages

Go to repository Settings → Pages → Select "main" branch → Save

Step 4: Access Your Site

Your site will be live at: https://YOUR_USERNAME.github.io/YOUR_REPO/
